Change text color in Sticky notes window - Synapbook

2018-05-15 1 min read
There are two ways to change the text color in the SynapBook sticky notes window: 1. Using the keyboard Move to the line you want to change and press the Change Color key. Blue - ctrl + alt + b Red - ctrl + alt + r * Press this key again to change to default color (black). Using the mouse Move to the line you want to change and press the right mouse button. In the SynapBook sticky notes window, there are two ways to check boxes: 1. Using the keyboard Move to the line you want to change and press the checkbox insert shortcuts alt + c. 2.
